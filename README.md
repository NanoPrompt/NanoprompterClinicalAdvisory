# Nanoprompter — Clinical Root-Cause Advisory (Demo)

A single-file HTML mockup of a hospital clinical advisory tool: dashboard, combined case intake/diagnosis panel, and an AI Intelligence Q&A panel with report upload/download.

**Status:** UI mockup with static sample data. No live model calls, no real patient data.

## Run locally
Just open `index.html` in a browser — no build step needed.

## Structure
- `index.html` — the entire app (HTML/CSS/JS in one file)
